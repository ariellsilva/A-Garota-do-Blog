
# 📺 Gossip Girl — Site Temático (HTML & CSS)

Este projeto consiste em um **site estático inspirado na série Gossip Girl**, desenvolvido com **HTML5 e CSS3**, com foco em **estrutura semântica, design visual, narrativa temática e responsividade**, sem utilização de JavaScript.

O site recria o clima do **Upper East Side**, utilizando textos narrativos no estilo da Gossip Girl, imagens temáticas e trilha sonora, proporcionando uma **experiência imersiva ao usuário**.

---

# 🌐 Páginas do Site

O projeto é composto por múltiplas páginas interligadas:

### Página Inicial (site.html)

Introdução narrativa no estilo Gossip Girl, com textos imersivos, imagens animadas (GIFs) e música de abertura da série.

### Personagens (personagens.html)

Página dedicada aos principais personagens da série, com descrições e identidade visual própria.

### Quem Sou Eu (quemSomos.html)

Apresentação pessoal da autora do projeto, formação acadêmica e objetivos profissionais.

### Contato (contato.html)

Informações de contato profissional, incluindo redes sociais e e-mail.

---

# 🎨 Tecnologias Utilizadas

### HTML5

* Uso de **tags semânticas** (`header`, `nav`, `main`, `section`)
* Estrutura clara e organizada
* Navegação entre páginas

### CSS3

* Estilização personalizada
* Layout centralizado com `max-width`
* Uso de **Flexbox** para alinhamento
* Responsividade com `width: 100%`
* Imagens adaptáveis (`max-width: 100%`)
* Identidade visual consistente

⚠️ Projeto totalmente **estático — não utiliza JavaScript**.

---

# 📱 Responsividade

O site foi desenvolvido para funcionar corretamente em:

📱 Smartphones
💻 Desktops
📟 Tablets

Utilizando práticas como:
